# Women's Day Tribute 💕

A beautiful, interactive Women's Day tribute website created with HTML, CSS, and JavaScript.

## 📋 Project Structure

```
ni-womens-day/
├── index.html       # Main HTML structure
├── style.css        # Styling and responsive design
├── script.js        # Interactive functionality
├── images/          # Image assets folder
│   └── white roses.avif
└── README.md        # This file
```

## 🎯 Features

- **Three-Page Interactive Experience**: Navigate through a beautiful story
  - Page 1: Introduction and Question 1
  - Page 2: Personal details and Question 2
  - Page 3: Final message and declaration
  
- **Responsive Design**: Works perfectly on all device sizes (mobile, tablet, desktop)
  
- **Animated Background Hearts**: Floating heart emojis that create a romantic atmosphere
  
- **Interactive No Button**: Can't say no! Playful button interaction on hover
  
- **Smooth Transitions**: Fade-in animations between pages
  
- **Scrollable Content**: Full scrolling support for longer content

## 🚀 How to Use

1. Open `index.html` in your web browser
2. Read the tribute and answer the questions
3. Click "Yes" buttons to proceed to the next page
4. Enjoy the farewell message!

## 💻 Technical Details

### HTML
- Semantic structure with proper comments
- Organized into three distinct page sections
- Accessible form elements (buttons, links)

### CSS
- Clean, well-organized styling with sections
- Mobile-first responsive design
- Custom animations (fade-in, floating hearts)
- Modern gradient backgrounds
- Proper contrast for readability

### JavaScript
- Pure vanilla JavaScript (no dependencies)
- Page navigation system
- Moving "No" buttons for playful interaction
- Smooth heart animation with requestAnimationFrame
- Lightweight and performant

## 📱 Browser Support

- Chrome/Edge (latest)
- Firefox (latest)
- Safari (latest)
- Mobile browsers (iOS Safari, Chrome Android)

## 🎨 Customization

To customize this tribute:

1. **Edit colors** in `style.css` (search for hex colors like `#ff8fab`)
2. **Change text** in `index.html` 
3. **Replace images** in the `images/` folder
4. **Modify animations** in `script.js`

## 📝 Notes

- All images should be placed in the `images/` folder
- Currently supports .avif format but can use .jpg, .png also
- The 3-page layout keeps the experience focused and interactive
